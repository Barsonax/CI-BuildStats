Buildstats.info

Build History Chart

Basic URL:
https://buildstats.info/{buildSystem}/chart/{account}/{project}

Supported values for {buildSystem}:

  • AppVeyor
  • TravisCI
  • CircleCI
Additional query parameters:
Branch

Displays the build history chart for a specific branch only:

branch={branch}
Build count

Displays the last X number of builds in the chart:

buildCount={number}
Exclude builds from a pull request

Displays the build history chart for builds which have not been triggered by a pull request:

includeBuildsFromPullRequest=false
Hide build time statistics

Hides the build time metrics (longest-, shortest- and average build time):

showstats=false
